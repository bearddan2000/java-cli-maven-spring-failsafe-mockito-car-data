# java-cli-maven-spring-failsafe-mockito-car-data

## Description
A POC for spring app. Testing done with failsafe
plugin using mockito framework.

## Tech stack
- java
- maven
  - spring
  - failsafe
  - mockito
    - junit5

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
